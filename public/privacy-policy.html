Digifall Privacy Policy

Introduction

This Privacy Policy describes how LLB LAB, a single-person indie development company, collects, uses, discloses and protects the information of users who use the Digifall application. Digifall is a free, open-source puzzle game available for download and play.

Information We Collect

  • Google Analytics

    We use basic Google Analytics to gather anonymized information about how users interact with Digifall. This information includes the number of game entries, country, screen resolution, and device type.

    This data is solely used to understand how people play Digifall and has no purpose of personalizing ads or using data for any other objective.

    Users have no option to opt out of Google Analytics tracking as basic configuration is used.

  • User Nicknames

    Users can choose a nickname, which is solely used for display purposes in the leaderboard. Nicknames are not stored on a server and are not used to identify users.

    Users can freely change their nickname at any time.

  • Game Data

    Digifall collects game data, such as the game start time, the sequence of moves, and the score.

    This data is stored on the user's device in an open format within IndexDB and is only accessible for viewing on the leaderboard.

    The data is only used to create a leaderboard and is not shared with third parties.

    Data is stored as long as the record holds one of the top 81 positions on the leaderboard, after which it can be deleted.

    Game data may be stored on other player's devices who participate in the P2P data exchange for the leaderboard.

Use of Information

  • Google Analytics

    Information gathered through Google Analytics is solely used to understand how people play Digifall and has no purpose of personalizing ads or using data for any other objective.

  • User Nicknames

    Nicknames are solely used for display purposes in the leaderboard.

  • Game Data

    Game data is only used to display on the leaderboard.

Disclosure of Information

  • Google Analytics

    We may disclose anonymized information gathered through Google Analytics to third parties, such as advertising partners, for analysis and service improvement purposes.

  • Game Data

    Game data is in the public domain and can be accessed by any user who chooses to download and analyze the game's open source code.

Information Security

Digifall does not store any user personal information on a server.

Game data is stored on the user's device in an open format within IndexDB.

Digifall utilizes standard security measures to protect data from unauthorized access, use, disclosure or alteration, but we are not responsible for the security of data stored on user devices.

User Rights

  • Access to Data

    Users can access game records found on the leaderboard.

  • Modification and Deletion of Data

    Users can modify their nickname but cannot delete game data.

  • Data Security

    Digifall is not responsible for the security of data stored on user devices.

Changes to this Privacy Policy

This Privacy Policy does not anticipate changes in the future.

Additional Information
